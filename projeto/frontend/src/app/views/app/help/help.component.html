Como utilizar o sistema

O FoodCare é uma plataforma intuitiva que conecta doadores e beneficiários de forma simples e eficiente. Para começar, você precisa criar uma conta escolhendo seu tipo de usuário.

Para Doadores: Você pode registrar alimentos disponíveis para doação, acompanhar o status das suas doações e gerar relatórios de impacto social.

Para Beneficiários: Você pode solicitar doações de acordo com suas necessidades, acompanhar suas solicitações e visualizar o histórico de recebimentos.
